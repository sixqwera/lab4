# Задание 1

## Задача 1

### Текст задачи

Дерево содержит целые числа. Увеличить каждый разряд на 1, если он не
содержит 9, иначе сохранить разряд неизменным. Например, 1296 ➔ 2397

### Алгоритм решения

Структура: Сначала мы создали тип Tree. Это рекурсивная заготовка: узел либо хранит число и ссылки на две ветки (лево/право), либо он пуст (Empty).
Печать: Функция recPrint обходит дерево методом In-order. Она сначала «ныряет» до упора влево, печатает число и уходит вправо. Так мы видим все узлы в консоли.
Трансформация: Функция treeMap — это наш «конвейер». Она создает копию дерева, применяя заданное правило к каждому числу в узлах.
Генерация: Через Random собирается небольшое случайное дерево из 5 узлов с числами от 0 до 9.
Логика: В main мы берем исходное дерево и инкрементируем каждое число на +1 (но с ограничением: если там уже 9, оставляем как есть).
