# Задание 1

## Задача 1

### Текст задачи

Дерево содержит целые числа. Увеличить каждый разряд на 1, если он не
содержит 9, иначе сохранить разряд неизменным. Например, 1296 ➔ 2397

### Алгоритм решения

Генерация дерева: Создается бинарное дерево из 5 узлов со случайными значениями от 0 до 9.
Вывод исходного дерева: Функция recPrint выполняет симметричный обход (левое поддерево → узел → правое поддерево) и выводит значения узлов.
Преобразование дерева: Функция Map применяет переданную функцию к каждому узлу:
Если значение узла = 9 → остается 9
Иначе → увеличивается на 1
Вывод результата: Выполняется симметричный обход измененного дерева с выводом новых значений.
